# ProjetoFinalEBAC - Predição de Fraudes em Transações de Cartão de Crédito
Predição de fraudes em transações de cartão de crédito com dataset altamente desbalanceado. O projeto compara técnicas de pré-processamento, reamostragem e ajuste de hiperparâmetros para detectar fraudes com mais eficiência, reduzindo falsos negativos e apoiando decisões mais seguras.
